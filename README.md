# CleanSwiftGenerator

Clean Swift Architecture(Uncle Bob's Clean Architecture applied to Swift language) file generator.

## Installation

-  Clone repository
-  From this repository:
    
    To install:
    ```
    $ make install
    ```

    To uninstall the Clean Swift Xcode templates, run:
    ```
    $ make uninstall
    ```
